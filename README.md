# README
These learning modules walks readers through the process of analyzing exoplanets via the transit method and explores how the exoplanet's environment contributes to biases in its detection and its overall evolution. This module is structured with the intetion of building students' toward new topics by drawing connections between novel and familiar topics. To do this, the module utilizes visual/graphical, textual, and mathematical descriptions to explain key concepts and engage students with all learning types. 

Overall, these learning modules will... 
1. Illustrate through multiple media how astrophysical phenomena such as exoplanets, stars, and galaxies can be studied through direct and indirect observations;
2. Connect prior knowledge to new learning by connecting familiar topics (our Earth) to unfamiliar topics (the Galactic thin and thick disk) to explore how different Galactic scales influence one another
3. Maximize transfer and generalization by emphasizing topics in data analysis and coding that can be applied to any discipline. In particular, this module emphasizes general python coding skills, including data visualization and data analysis. 

Upon completion of these modules, readers will... 
1. Have a conceptual understanding of the study of exoplanets and their host stars can inform our understanding of our galaxy and how our understanding of our galaxy can inform our understanding of exoplanets and their host stars;
2. Be able to work through a simple data analysis process that takes reduced TESS data and outputs key qualities of a transiting exoplanet
3. Gain skills in coding and data analysis that they can apply to other areas of interest (e.g., time series analysis)

## Teaching this module 
These modules are geared toward an introductory astrophysics student, with minimal, but not extensive coding knowledge. The idea behind this module is to allow an introductory student to draw comparisons between familiar and unfamiliar concepts and use those comparisons to build understanding.
